# ✦ LOGO SUITE DESIGN DIRECTION — REVISED ✦
### For: **ROSEMARY LEGOH** and **LEGOH ATELIER**
### Aesthetic Mandate: **Modernist Architecture × Abstract Indonesian Wastra**
### Deliverable Format: **3 Identity Concepts**, each including:
1. Primary Logotype with structured custom typography  
2. Abstract Monogram (The Mark) for hardware and tags  
3. Minimalist **Tanah & Laut** color palette  
4. Design rationale with a clear **Architectural Signature**

---

## CREATIVE POSITION

The original strategic direction is strong: the brand should feel like **quiet luxury with structural intelligence** rather than decorative fashion.  
However, for execution clarity, the logo suite should be organized as **three complete identity systems**, not as a larger pool of uneven ideas.  

The three concepts below are the most strategically relevant for a Southeast Asian designer brand rooted in:
- **architectural restraint**
- **material intelligence**
- **Indonesian wastra memory**
- **a premium but modern feminine authority**

Each concept is designed to work for **both top recommended names**:

- **ROSEMARY LEGOH** → more personal, sovereign, designer-led
- **LEGOH ATELIER** → more institutional, crafted, studio-led

---

# CONCEPT 01 — THE FOLD LINE
## Architectural Signature: **The Cantilever**

This concept is the strongest choice if the brand wants to express **structural innovation** most clearly.  
Its key idea is **controlled asymmetry**: a composition that feels balanced not because it is centered, but because it is intelligently supported.

This concept translates especially well if the fashion collection will build around a signature construction detail like the **RL Fold / asymmetric collar**.

---

## 1) PRIMARY LOGOTYPE

### A. Version for **ROSEMARY LEGOH**

```text
ROSEMARY
──────────────
        LEGOH
```

**Typography Direction**
- Custom serif with architectural proportions
- High contrast, but not fragile
- Slightly condensed capitals
- Wide tracking for **ROSEMARY**
- Heavier optical weight for **LEGOH**
- Horizontal line acts as a structural beam

**Design Logic**
- “ROSEMARY” feels lighter, more refined, more personal
- “LEGOH” lands with authority
- The line between them becomes the key gesture: a typographic fold, beam, and seam all at once

### B. Version for **LEGOH ATELIER**

```text
LEGOH
──────────────
   ATELIER
```

**Typography Direction**
- Same custom serif family for brand coherence
- “LEGOH” set larger and more architectural
- “ATELIER” smaller, letterspaced, quieter
- The rule line becomes the threshold between brand and practice

**Design Logic**
- “LEGOH” is the house
- “ATELIER” is the making space beneath it
- Reads as premium, structured, and timeless

---

## 2) ABSTRACT MONOGRAM — THE MARK

### Mark Structure
A triangular or folded-line monogram inspired by:
- the **tumpal** motif in wastra
- a folded collar edge
- a cantilever support
- a structural roofline

### Option A — for **ROSEMARY LEGOH**
- Abstract **R/L** hidden inside a folded triangular geometry
- Best for:
  - metal hardware
  - woven labels
  - wax seals / embossed packaging

### Option B — for **LEGOH ATELIER**
- Abstract **L/A** formed from a folded threshold shape
- Slightly more geometric and institutional
- Better for:
  - signage
  - garment tags
  - buttons and zip pulls

**Monogram Principle**
The mark should not read like a literal monogram first.  
It should read first as a **luxury symbol**, and only second as letters.

# CONCEPT 02 — THE THRESHOLD
## Architectural Signature: **The Doorframe**

This concept is about **entry, framing, and sacred boundary**.  
It references both:
- the logic of modernist architectural portals
- and Indonesian threshold symbolism: *gapura*, *candi bentar*, and textile borders

It is the best concept if the brand wants to feel like a **house** or **atelier with spatial authority**.

---

## 1) PRIMARY LOGOTYPE

### A. Version for **ROSEMARY LEGOH**

```text
│ ROSEMARY
│ LEGOH
```

or a more refined typographic variant:

```text
│ ROSEMARY
│ ─────────
│   LEGOH
```

**Typography Direction**
- Tall serif capitals with disciplined spacing
- Vertical rule as permanent signature element
- The vertical line is not decoration; it is architecture

**Design Logic**
- The name appears as if installed beside a structural column
- Suggests permanence, entrance, and composure

### B. Version for **LEGOH ATELIER**

```text
│ LEGOH
│ ─────────
│   ATELIER
```

**Typography Direction**
- “LEGOH” bold and close to the vertical rule
- “ATELIER” lighter, inset, more spacious
- Feels like a plaque on a building or atelier door

**Design Logic**
This is especially strong for **Legoh Atelier** because “atelier” implies a place of making.  
The logo literally behaves like an entrance.

---

## 2) ABSTRACT MONOGRAM — THE MARK

### Mark Structure
A rectilinear monogram built from:
- two vertical posts
- one threshold line
- negative space as doorway

### Option A — for **ROSEMARY LEGOH**
- Interlocked **R/L** reduced to a floor-plan logic
- Should feel like:
  - a room
  - a column
  - a carved panel

### Option B — for **LEGOH ATELIER**
- Interlocked **L/A** as an open gateway
- Most suitable as a cast metal mark
- Strong when embossed small

**Monogram Principle**
This mark should rely heavily on **negative space**.  
It must feel architectural before alphabetical.

# CONCEPT 03 — THE LOOM GRID
## Architectural Signature: **The Structural Grid**

This concept is the most intellectually driven.  
It treats typography as a weaving system — a grid of measured intervals, like an elevation drawing or loom plan.

It references:
- the Miesian modernist grid
- woven textile planning
- repetition and module in Indonesian craft

This is the strongest concept if the brand wants to feel more **design-literate, editorial, and contemporary**.

---

## 1) PRIMARY LOGOTYPE

### A. Version for **ROSEMARY LEGOH**

```text
R O S E M A R Y
| | | | | | | |
L E G O H
```

**Typography Direction**
- Geometric custom sans or low-contrast serif hybrid
- Sharp joints and measured spacing
- Vertical strokes between the two lines suggest warp threads

**Design Logic**
- The name is visually woven
- The identity becomes a textile system rather than just a wordmark

### B. Version for **LEGOH ATELIER**

```text
L E G O H
| | | | |
A T E L I E R
```

or a more resolved version:

```text
LEGOH
······
ATELIER
```

with modular spacing and blueprint logic.

**Typography Direction**
- More modular than expressive
- Built for editorial use, labels, and graphic systems
- Works beautifully in lookbooks and campaign layouts

---

## 2) ABSTRACT MONOGRAM — THE MARK

### Mark Structure
A modular monogram based on:
- interlocking vertical/horizontal members
- floor-plan geometry
- weaving frame logic

### Option A — for **ROSEMARY LEGOH**
- Abstract **R/L** as a structural plan
- Best when reduced to a square or rectangular stamp

### Option B — for **LEGOH ATELIER**
- Open geometric bracket or frame with a central point
- Reads like:
  - a loom bracket
  - blueprint notation
  - selvedge marker
  - architectural room label

**Monogram Principle**
This is the least emotional and most cerebral mark.  
It is ideal if the brand wants a more editorial, art-direction-forward identity.

**Strength**

It may not be the easiest for mass recognition, but it is the strongest for building a **cult design identity**.

# FINAL CREATIVE DIRECTOR RECOMMENDATION

If choosing only one direction per name:

## For **ROSEMARY LEGOH**
**Recommended Concept: 01 — The Fold Line**

Why:
- It feels personal but not sentimental
- It carries quiet luxury beautifully
- It aligns best with the idea of a signature structural garment detail
- It gives the brand the most elegant long-term recognition system

## For **LEGOH ATELIER**
**Recommended Concept: 02 — The Threshold**

Why:
- It makes “Atelier” feel spatial and meaningful
- It is stronger for signage, packaging, and a future physical boutique
- It best expresses the idea of a crafted house of design

## If the brand wants one shared system for both names
Choose **Concept 03 — The Loom Grid** as a supporting graphic language, not necessarily as the master logo.

That is the most useful hierarchy:
- **Master Logo Direction** = Fold Line or Threshold
- **Graphic System Language** = Loom Grid
